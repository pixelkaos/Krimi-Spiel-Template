
# Gemeinsames Vokabular

Kanonische Begriffe, damit alle Agenten konsistent formulieren. Fallspezifische Namen folgen mit der
[[Fallbibel]] (Greenfield: noch offen).

## Spielmechanik / Design
- **Geschlossener Kreis (Closed Circle):** begrenzte Verdächtigenmenge (3–8), isoliertes Umfeld.
- **MOCA:** Motiv · Gelegenheit · Beziehung (Connection) · Alibi — Raster je Verdächtige:r.
- **Truth Map / Wahrheitsgitter:** mechanisch prüfbare Lösungslogik (eindeutig erzwungen).
- **Knoten:** Ermittlungsstation/Ort. Rollen: **H** = Pflicht, **S** = Bonus, **R** = Red Herring.
- **Umschlag (T01–T24):** physisches Tagespaket (Koordinate/Label, Zeitkosten, Hinweise, führt-zu).
- **Branch-and-Bottleneck:** freie Reihenfolge mit Pacing-Gates (Bottlenecks z. B. via Zeitungen).
- **Zeitbudget:** „Ermittlungsstunden" pro Kalendertag (begrenzt Umschlag-Öffnungen).
- **Roter Hering (Red Herring):** falsche Spur — braucht **fairen Exit** + lehrt etwas.
- **Hinweis-Leiter (Hint Ladder):** 3 gestaffelte, spoilerarme Tipps pro Rätsel.
- **Liar-Mechanik:** Täter:in lügt, unbescholtene Zeug:innen sagen die Wahrheit (klar markiert).

## Stimmen (siehe `kb/styles/`)
- **Spielleiter-Wir:** die beiden Spielleiter als Erzähler-„wir", adressieren „Du/Ihr".
- **Dokument:** diegetische Artefakte (Zeitung/Brief/Protokoll/Aktennotiz).
- **Erzähler 3. Person:** neutral-atmosphärische Verbindungstexte.
