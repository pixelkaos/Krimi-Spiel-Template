# Schreibproben

Lege hier echte Textproben ab (eure früheren Texte, Wunsch-Vorbilder, oder die ersten Umschlag-Entwürfe).
Sie sind die Grundlage für `/style-analysis`, das daraus die Stil-Dateien in `kb/styles/` verfeinert.

Noch leer (Greenfield). Sobald 1–2 echte Texte pro Stimme existieren, Stilanalyse anstoßen.
