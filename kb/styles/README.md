# Stimmen-System (Voice)

Drei umschaltbare Erzählperspektiven. Jede Prosa-Notiz wählt ihre Stimme im Frontmatter
(`stimme: spielleiter | dokument | erzaehler3`). Grundton überall: **gemütlich-ironisch, SHCD-nah,
nie brutal, fair-play; Anrede Du/Ihr; Deutsch.**

| Stimme | Datei | Einsatz |
|---|---|---|
| Spielleiter-Wir | `spielleiter-wir.md` | Begrüßung, Tagesintros, Hinweis-Leiter, Auflösung |
| Dokument | `dokument.md` | Zeitung, Brief, Protokoll, Aktennotiz (Beweis-Kern) |
| Erzähler 3. Person | `erzaehler3.md` | verbindende Szenen / Ortsbeschreibungen |

> **Stand:** Starter-Dateien mit Voice-Zielen. Sobald die ersten echten Texte geschrieben sind, mit
> `/style-analysis` aus den Proben (`kb/samples/`) zu vollwertigen Stil-Dateien verfeinern.
